Dfinitions

 

GVLOGO travaille essentiellement partir de mots et de listes. Ce chapitre se propose dtudier les mots.

Un mot est une suite quelconque de caractres.

Un mot est dlimit par un espace, les signes [, ], (, ). Ces cinq caractres particuliers sont appels des dlimiteurs : ils indiquent GVLOGO comment sparer les lments du langage.

Il existe un mot particulier qui ne comprend aucun caractre : on l'appelle le mot vide. On le reprsente par un guillemet anglais suivi d'un espace.

Exemple :
